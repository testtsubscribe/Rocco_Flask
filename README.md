# ROCCO Corporate Website

Modern corporate website for ROCCO, specializing in professional cutting and grinding discs. The website features a Flask backend, a SQLite database for content management, and full support for three languages.

## Key Features
- **Multi-language Support**: Azerbaijani (default), Russian, and English.
- **Dynamic Content**: All website text and company information are fetched from a SQLite database.
- **Premium Design**: Modern industrial aesthetic with a dark theme, high-quality visuals, and responsive layout.
- **Interactive UI**: Smooth scrolling, language switcher, and a contact form.

## How to Run
1. Ensure you have Flask installed: `pip install flask`
2. Run the application: `python3 app.py`
3. Open `http://127.0.0.1:5000` in your browser.
